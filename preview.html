Method 1: VS Code Live Server (Easiest - No Installation Needed)

  1. Open VS Code (or install it from code.visualstudio.com)
  2. Install the "Live Server" extension (search for it in Extensions)
  3. Open this folder in VS Code
  4. Right-click on index.html
  5. Select "Open with Live Server"
  6. Your browser will open automatically!

Method 2: Install Xcode Command Line Tools (One-Time Setup)

If you want to use Python, you'll need to install developer tools first:

  1. Run this command in Terminal:
    xcode-select --install
  2. Follow the installation dialog (this downloads ~500MB)
  3. After installation, try the Python server again:
    python3 -m http.server 8000

Method 3: Use Node.js (If You Have It)

If you have Node.js installed:

  1. Install http-server: npm install -g http-server
  2. Run: http-server -p 8000
  3. Open: http://localhost:8000
